Git命令可以很方便地重命名本地分支
$ git branch -m <old_branch_name> <new_branch_name>

删除远程待修改的分支名（注意：git branch -d YourBranchName只能删除本地的branch）
$ git push origin :YourBranchName

最后将本地的新分支名push到远程：
$ git push origin
